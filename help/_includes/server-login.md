1. Meld u aan bij uw server.
1. Naar de [eigenaar van bestandssysteem](../installation/prerequisites/file-system/overview.md).
1. Ga naar de map waarin u de toepassing hebt gekloond. Bijvoorbeeld:

   ```bash
   cd /var/www/magento2
   ```
