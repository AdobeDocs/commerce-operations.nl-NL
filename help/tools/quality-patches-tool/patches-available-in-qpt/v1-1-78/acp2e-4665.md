# ACS2E-4665: De producten van kinderen van configureerbare producten met video&#39;s in de productgalerieën verschijnen niet wanneer gevraagd door REST API

De ACS2E-4665-patch verhelpt het probleem dat onderliggende producten van een configureerbaar product niet door de REST API worden geretourneerd wanneer een video in de productgalerie wordt opgenomen. Deze patch is beschikbaar wanneer [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78 wordt geïnstalleerd. De patch-id is ACP2E-4665. Dit probleem wordt volgens de planning opgelost in Adobe Commerce 2.4.9.

## Betrokken producten en versies

**het flard wordt gecreeerd voor de versie van Adobe Commerce:**

* Adobe Commerce (alle implementatiemethoden) 2.4.8-p3

**Compatibel met de versies van Adobe Commerce:**

* Adobe Commerce (alle implementatiemethoden) 2.4.8 - 2.4.8-p4

>[!NOTE]
>
>De patch kan van toepassing worden op andere versies met nieuwe [!DNL Quality Patches Tool] versies. Als u wilt controleren of de patch compatibel is met uw Adobe Commerce-versie, werkt u het `magento/quality-patches` -pakket bij naar de meest recente versie en controleert u de compatibiliteit op de [[!DNL Quality Patches Tool] : Zoek naar de pagina van flarden ](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Gebruik de patch-id als een zoekwoord om de patch te zoeken.

## Probleem

De REST API keert geen kindproducten van een configureerbaar product terug wanneer een video in de productgalerij aanwezig is.

<u> Stappen om </u> te reproduceren:

1. Voeg een video aan een configureerbaar kindproduct van het product toe.
1. Verzend een verzoek naar het volgende REST API-eindpunt:
   `rest/v1/configurable-products/<PARENT_CONFIGURABLE_PRODUCT_SKU>/children`.

<u> Verwachte resultaten </u>:

De REST API retourneert informatie over het product.

<u> Ware resultaten </u>:

De REST API retourneert de volgende fout:

```
Error: Call to a member function getVideoProvider() on null in vendor/magento/module-product-video/Model/Product/Attribute/Media/ExternalVideoEntryConverter.php:87" exception.
```

## De patch toepassen

Om individuele flarden toe te passen, gebruik de volgende verbindingen afhankelijk van uw plaatsingsmethode:

* Adobe Commerce of Magento Open Source ter plaatse: [[!DNL Quality Patches Tool] > Gebruik ](/help/tools/quality-patches-tool/usage.md) in de [!DNL Quality Patches Tool] gids.
* Adobe Commerce op cloudinfrastructuur: [ Verbeteringen en Patches > pas Patches ](https://experienceleague.adobe.com/docs/commerce-cloud-service/user-guide/develop/upgrade/apply-patches.html) in Commerce op de gids van de Infrastructuur van de Wolk toe.
